FAQ

Problemas de Internet:

Há conexão de internet no computador do cliente?

  • Executar solução de problema do sistema operacional
  • Reiniciar o modem
  • Ligar para a operadora de internet

Hardware:

Existe algum dano externo na instalação?

  • Conferir se o led de alimentação está aceso
  • Checar as conexões de alimentação do Arduino
  • Checar se as baterias estão com carga
  • Checar se o Arduino está danificado

Arduino está ligando, mas os dados não estão sendo visualizados no dashboard/site

  • Conferir se o sensor está operacional
  • Conferir cabeamento de conexão do sensor do Arduino

Container:

Container não está mantendo a temperatura?

  • Enviar o container à Vaccinus para análise

Container está danificado?

  • Enviar o container à Vaccinus para análise
